# Cancelling an order

Whether an order can be cancelled depends entirely on whether it has entered the shipping process yet — not on how much time has passed since it was placed.

## Before shipping

If the order hasn't entered the shipping process, it can be cancelled directly through order history — select the item(s), choose a cancellation reason, and confirm. No approval is needed for items fulfilled directly, and the customer is never charged for a cancelled order. A confirmation is sent to the account email once it's processed.

Orders sold and shipped by a third-party seller can usually still be cancelled within about one business day of being placed, but once that seller has moved the order into fulfillment, they have to approve the cancellation themselves rather than it happening automatically.

## After shipping

Once an order has entered the shipping process, it can no longer be cancelled directly. At this point the options are:
1. Let the delivery complete, then start a standard return if unwanted
2. Refuse the delivery at the door — the carrier will log this, and the charge is typically refunded within a few days once Amazon confirms the refusal
3. Use the online returns center once it's delivered, if refusing delivery wasn't possible

There's no way to guarantee a cancellation goes through once shipping has started — it depends on how far along the package already is, not on policy negotiation.

## Cancelling only part of a multi-item order

Customers can cancel individual items within an order as long as those specific items haven't entered shipping yet, even if other items in the same order have already shipped. Each item in an order tracks its own fulfillment status independently.

## Changing the delivery address

Once an order is confirmed, the delivery address generally can't be edited directly. If the order hasn't shipped yet, the practical option is to cancel and reorder with the correct address. If it has already shipped, an address change isn't something that can be promised — this should be escalated rather than guaranteed to the customer.
